About us

A-Mapoteca is a collection of maps created by the Alfobre team.

Alfobre is portuguese for seedbed. It's also a collective devoted to the creation and development of projects, strategies and products in the Heritage field.

This website is powered by Jekyll and a modified version of the Adam Blog theme. If you're interested in checking or using our version, please visit the GitHub repository. The original theme is licensed under the GNU GPL 3 and so is A-Mapoteca.
